## 📚 Course Overview

VibeCoding is an 8-week intensive web development course that teaches students to build modern websites using **HTML, CSS, JavaScript**, and **GitHub Copilot** as an AI pair programmer. Students learn professional workflows including version control, debugging, and deployment.

### Learning Outcomes
- Build responsive websites with HTML5, CSS3, and JavaScript
- Use GitHub Copilot for AI-assisted coding
- Debug code systematically with AI tools
- Deploy projects via GitHub Pages
- Collaborate using git and GitHub
- Apply prompt engineering for effective AI assistance

## 🗓️ Course Structure

| Week | Topic | Assignments |
|------|-------|-------------|
| 1 | Setup & AI Fundamentals | Day 1 Onboarding, Day 2 Debug Exercise |
| 2 | HTML & CSS Basics | HTML Fundamentals, CSS Styling |
| 3 | JavaScript Fundamentals | JS Basics |
| 4 | **Project 1: Personal Portfolio** | Portfolio website (GitHub Classroom) |
| 5 | Advanced JavaScript | Event handling, DOM manipulation |
| 6 | APIs & Data | Fetch API, JSON, async/await |
| 7 | **Project 2: Interactive App** | Web app with API integration (GitHub Classroom) |
| 8 | Final Presentations | Demo & code walkthrough |

## 🤖 AI Tools Integration

This course emphasizes **AI-assisted development** using:

- **GitHub Copilot** - AI pair programmer for code completion
- **Copilot Chat** - Conversational AI for explanations and debugging
- **Prompt engineering** - Crafting effective requests to AI
- **AI debugging workflows** - Systematic problem-solving with AI assistance

Students learn to use AI as a **learning amplifier**, not a replacement for understanding.

## 👥 For Teachers

### Getting Started
1. Review [TEACHER-MANUAL.html](https://kinkaidemsi.github.io/vibecoding-2026-materials/TEACHER-MANUAL.html)
2. Follow [INFRASTRUCTURE-SETUP.md](INFRASTRUCTURE-SETUP.md) to configure GitHub Classroom
3. Set up Google Classroom with assignment links
4. Upload handouts (Day 1, AI Cheat Sheet) to Google Classroom
5. Test template repository and Codespaces

### Grading Rubrics
Detailed rubrics for all assignments are in the Teacher Manual, including:
- Setup completion (10 points)
- Debug exercise (15 points)
- Weekly assignments (20-30 points each)
- Project 1: Portfolio (100 points)
- Project 2: Interactive App (150 points)
- Final presentation (50 points)

### Office Hours & Support
- **Student help:** Daily standup + 1-on-1 debugging sessions
- **Parent communication:** Email templates in Teacher Manual
- **Technical issues:** Troubleshooting guide in Teacher Manual
