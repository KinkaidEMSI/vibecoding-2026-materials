AI Guide

How to get useful code out of AI. This is the whole skill.

YOUR TWO TOOLS

Gemini Code Assist

Lives inside your Codespace. Right panel after Day 1 setup. Writes code directly into your files. This is your main tool.

AI Studio  backup

Browser tab at aistudio.google.com. Use when Gemini is acting weird or you want to think through a problem before coding.

OPENING GEMINI IN YOUR CODESPACE

1

Find the Gemini icon

Look for a sparkle or diamond icon in the left Activity Bar. Click it.

2

Sign in with Google

Click Sign in with Google → use your personal Google account. School Google accounts sometimes have restrictions.

3

Move it to the right panel

Right-click the Gemini icon → Move to Secondary Side Bar. Now you see code on the left and AI chat on the right at the same time.

PROMPTS THAT WORK

Be specific. Describe what you want to see.

You don’t need to know HTML or CSS terms. Describe the visual result.

✗ “Make my website better”

✓ “Add a navigation bar at the top with links to Home, About, and Projects. Use the same dark background. Make the links white and bold.”

One thing at a time

Ask AI to change one section at a time. If you ask for 10 things at once, something will break. Build up piece by piece, checking after each change.

Tell AI what you already have

When asking for changes, describe the existing page first.

✓ “I have a page with a dark background, a big white heading, and a short paragraph. Add a grid of 3 cards below the paragraph. Each card should have a title and 2 sentences. The topic is my hobbies.”

Describe style in plain English

✓ “Dark navy background, white text, one bright accent color, big headings, lots of spacing”

✓ “Bright and colorful, like a game UI, with a pixel font for headings”

✓ “Clean and minimal, lots of whitespace, black and white with one yellow accent”

WHEN AI GIVES YOU GARBAGE

Tell it exactly what’s wrong

“The font is too small and the background is the wrong color. Make the heading 64px. Change the background to #1a1a2e and the text to white.”

Ask it to fix one specific thing

“The navigation is overlapping the main content. Fix it so the nav stays at the top and the content starts below.”

Paste your code and ask what’s wrong

Copy your HTML, paste it in the chat: “This isn’t doing what I want. [Describe the problem]. What’s wrong and how do I fix it?”

Start over with a better prompt

Sometimes it’s faster to write a clearer prompt from scratch than to fix bad output. That’s completely normal.

WHAT AI IS ACTUALLY GOOD AND BAD AT

  • ✓ HTML structure, CSS layout and styling
  • ✓ Quizzes, counters, clickers, random generators
  • ✓ Animations and hover effects
  • ✓ Explaining your existing code
  • ✓ Fixing specific bugs when you describe them clearly
  • ✗ Multi-page interactions that require a server
  • ✗ Connecting to external APIs without extra setup
  • ✗ Pixel-perfect layouts from a vague description — be specific
