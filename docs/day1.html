Day 1 — Setup

Jun 8 · 40 minutes · Do these steps in order

⚠ BEFORE CLASS — do this tonight or tomorrow morning:
Go to github.com and create a free account using your school email. It takes 2 minutes and saves 10 minutes of class time.

PART 1 — GITHUB (10 min)

01

Create a GitHub account

Go to github.com → Sign up → school email → free plan. Already have one? Skip this step.

02

Accept the GitHub Classroom invite

Open this link and sign in. This creates your personal project repo.

https://classroom.github.com/a/xuJPS0u1

03

Open your Codespace

In your new repo → click Code → Codespaces → Create codespace on main.

Wait ~90 seconds. VS Code opens in your browser with everything installed and ready.

PART 2 — AI SETUP (5 min)

04

Sign into Gemini

Find the Gemini icon (sparkle/diamond) in the left sidebar → click it → Sign in with Google → use your personal Google account.

05

Move Gemini to the right panel  optional

Right-click the Gemini icon → Move to Secondary Side Bar. AI chat on the right, your code on the left.

PART 3 — BUILD YOUR FIRST PAGE (20 min)

06

Start the live preview

Click index.html in the file explorer → right-click inside the editor → Open with Live Server. A preview opens. Every save = instant update.

07

Type your first AI prompt

Open the Gemini chat panel and type something like this:

“Build me a personal webpage. My name is [your name]. I’m interested in [2–3 things you like]. Make it look modern with a dark background and bold typography.”
08

Paste the code and watch it appear

Copy all the HTML Gemini gives you → click inside index.html → Select All (Ctrl+A) → Paste (Ctrl+V) → Save (Ctrl+S).

Watch it appear live. Ask AI to change colors, fonts, add sections. Iterate.

PART 4 — SAVE & PUSH (5 min)

09

Push your work to GitHub

Open the terminal: Terminal → New Terminal (or press Ctrl+`). Run these 3 commands:

git add .
git commit -m "day 1 first page"
git push

You will do this at the end of every single class. It saves and publishes your work.

10

Turn on GitHub Pages  if time

Go to your repo on github.com → Settings → Pages → Source: Deploy from branch → main → / (root) → Save.

Wait ~1 min → your site is live at [username].github.io/[repo-name]. One-time setup only.

✓ Done for Day 1. You have a page, you pushed it, you know the 3 commands. Everything else builds on this.
