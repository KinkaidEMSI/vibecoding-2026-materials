Git Cheat Sheet

3 commands. End of every class. Every time.

SAVE & PUBLISH YOUR WORK

git add . Stage everything you changed
git commit -m “...” Describe what you built
git push Send it to GitHub & update your live site
Write a real commit message.
✓ “added music section and dark mode toggle”
✗ “stuff”    ✗ “aaa”    ✗ “update”

OPEN A TERMINAL IN YOUR CODESPACE

Menu: Terminal → New Terminal — or press Ctrl + `

TURN ON GITHUB PAGES — ONE TIME ONLY

  • 1Go to your repo on github.com
  • 2Click Settings → Pages
  • 3Source: Deploy from branch → branch: main → folder: / (root) → Save
  • 4Wait ~1 minute → your site is live at [username].github.io/[repo-name]

After this, every git push automatically updates your live site. Never touch Pages settings again.
