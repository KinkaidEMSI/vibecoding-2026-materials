Setup — do these in order
  1. 1
    Create a GitHub account Use your school email. Free plan. Write down your username — you'll use it every day. github.com/signup
  2. 2
    Tell Mr. V your username Type it in the Google Form on the board — or raise your hand. You'll get an email invite to the class organization within 2 minutes.
  3. 3
    Accept the GitHub Classroom invite Check your school email. Click the link. Accept the assignment. GitHub creates your personal repo automatically.
  4. 4
    Open your Codespace In your repo, click the green Code button → Codespaces → Create codespace on main. First load takes ~90 seconds. This is normal. Don't close the tab.
  5. 5
    Open index.html It's already there. Click it in the left sidebar. You'll see starter code.
  6. 6
    Change the title to your name Find the line that says <h1>Your Name</h1> and replace it. That's your first code edit.
  7. 7
    Save & preview Press Ctrl + S (or Cmd + S). Click Go Live in the bottom bar. A preview tab opens.

The only 3 Git commands you need (learn these today)
  • git add . Stage everything you changed. The dot means "all files." Run this first.
  • git commit -m "msg" Save a snapshot with a message. Replace msg with what you did — e.g. "added header" or "fixed navbar color". Be specific.
  • git push Send your snapshot to GitHub. This is what triggers GitHub Pages to update your live site. Always push before leaving class.
